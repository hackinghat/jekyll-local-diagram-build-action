Mermaid

This is a showcase for Mermaid diagrams to show what is possible with Jekyll Local Build Action and to act as an integration test for Jekyll Local Diagram

All diagrams are generated images, and can be shared as files.

Class Diagram 

{% mermaid %}
classDiagram
direction RL
class Student {
    -idCard : IdCard
}
class IdCard{
    -id : int
    -name : string
}
class Bike{
    -id : int
    -name : string
}
Student "1" --o "1" IdCard : carries
Student "1" --o "1" Bike : rides
{% endmermaid %}
